    Nudity Detection

    POST

    Summary: Nudity Detection

    Description: Inappropriate content classifier is a powerful tool to filter out pornographic images from social media feeds, forums, photo sharing apps, etc. ParallelDots inappropriate content classifier can filter such content and help build a safer platform for your community.

    require 'paralleldots'
path = "/home/my_computer/Downloads/image_1.jpg"

puts( nsfw( path ) )

    import paralleldots

path = "/home/my_computer/Downloads/image_1.jpg"
print( paralleldots.nsfw( path ) )

    
To send a image file.

curl -X POST --header 'Content-Type: multipart/form-data' --header 'Accept: application/json' {"type":"formData"} -F "file=@/home/my_computer/Downloads/image_1.jpg" -F "api_key=ABCdef123MNO456PQR789xyz" 'https://apis.paralleldots.com/v3/nsfw'

To send a image url.

curl -X POST --header 'Content-Type: multipart/form-data' --header 'Accept: amData"} -F "url=https://www.google.com/image-400184.jpg" -F "api_key=ABCdef123MNO456PQR789xyz" 'https://apis.paralleldots.com/v3/popularity'

    import paralleldots.ParallelDots;

ParallelDots pd = new ParallelDots();
String path_to_image="<path_to_image>"
String nsfw = pd.nsfw(path_to_image);
System.out.println(nsfw);


    require(__DIR__ . '/vendor/paralleldots/apis/autoload.php');
$path_to_image = "<path_to_image>";
echo nsfw($path_to_image);


    path = "/home/my_computer/Downloads/image_1.jpg"
String nsfw = pd.nsfw(path_to_image);
Console.WriteLine(nsfw);

    The above command returns JSON structured like this:

    {
  "output": "not safe to open at work", 
  "prob": "0.9999975562095642"
},
  "usage": "By accessing ParallelDots API or using information generated by ParallelDots API, you are agreeing to be bound by the ParallelDots API Terms of Use: https://www.paralleldots.com/terms-and-conditions"
}

    HTTP Request

    ***POST*** /nsfw

    Parameters

    Either file or url is mandatory , do not send both parameters.

    Name Located in Description Required Type
    file query Pass a image file Yes file
    url query Pass url of image file Yes string
    api_key query Apikey Yes string

    Responses

    Name Description Type
    output If the imgage safe to open at work or not
    prob The score of above output. It lies between 0 to 1 . Higher score states the higher confidence score of the output.
    Code Description
    200 Successful response
    304 Not Modified
    500 Internal Server Error

    Virality Detection

    POST

    Summary: Virality Detection

    Description: Virality Detection API gives a score to images based on their potential to go viral on the Internet. The AI model is trained using a super deep neural network on a massive corpus of images and their likes count as crawled from the open web. The score that you get as an output is the virality score of the input photo out of a maximum score of 100. 

    require 'paralleldots'

path = "/home/my_computer/Downloads/image_1.jpg"

puts( popularity( path ) )

    import paralleldots

path = "/home/my_computer/Downloads/image_1.jpg"
print( "Popularity" )
print( paralleldots.popularity( path ) )

curl -X POST --header 'Content-Type: multipart/form-data' --header 'Accept: application/json' {"type":"formData"} -F "file=@/home/my_computer/Downloads/image_1.jpg" -F "api_key=ABCdef123MNO456PQR789xyz" 'https://apis.paralleldots.com/v3/popularity'

To send a image url.

curl -X POST --header 'Content-Type: multipart/form-data' --header 'Accept: amData"} -F "url=https://www.google.com/image-400184.jpg" -F "api_key=ABCdef123MNO456PQR789xyz" 'https://apis.paralleldots.com/v3/popularity'

    HTTP Request

    ***POST*** /popularity

    Parameters

    Either file or url is mandatory , do not send both parameters.

    Name Located in Description Required Type
    file query Pass a image file Yes if url not provided file
    url query Pass url of image file Yes if file not provided string
    api_key query Apikey Yes string

    Responses

    Name Description Type
    Popular The confidence score of the virality of the image. It lies between 0 to 1 . Higher score states the higher confidence score of the output.
    Not Popular The confidence score of the non-virality of the image. It lies between 0 to 1 . Higher score states the higher confidence score of the output.
    Code Description
    200 Successful response
    304 Not Modified
    500 Internal Server Error
